# 📊 Dashboard de Vendas com Streamlit

Este repositório foi criado para o **Exercício 2** da disciplina. Trata-se de uma aplicação web interativa desenvolvida em Python utilizando a biblioteca **Streamlit** para visualização e análise de dados de vendas.

## 🚀 Link do Aplicativo
O dashboard está publicado e pode ser acessado em tempo real através do link abaixo:
👉 **[CLIQUE AQUI PARA ACESSAR O DASHBOARD](https://seu-link-do-streamlit.streamlit.app)**

---

## 🛠️ Tecnologias Utilizadas

- **Python** (Linguagem de programação principal)
- **Streamlit** (Framework para criação da interface web)
- **Pandas** (Manipulação e tratamento dos dados)
- **Plotly Express** (Criação de gráficos interativos)

## 📁 Estrutura do Projeto

- `streamlit_app3.py`: Código-fonte da aplicação Streamlit.
- `vendas_dashboard.csv`: Base de dados utilizada pelo dashboard.
- `requirements.txt`: Arquivo com as dependências necessárias para o deploy no servidor.

---

## 💻 Como rodar o projeto localmente

Se desejar executar este projeto na sua máquina:

1. Clone o repositório:
   ```bash
   git clone [https://github.com/karolaineponci/karolaine-ponci-streamlit.git](https://github.com/karolaineponci/karolaine-ponci-streamlit.git)

2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
3. Execute o Streamlit:
   ```bash
   streamlit run streamlit_app3.py
